Flex - Direction

Utilizar o flex-direction: define a direção com que os elementos irão seguir.

EXEMPLO:

FLEX-DIRECTION: ROW (modo padrão quando coloca apenas display: flex)

FLEX-DIRECTION: ROW-REVERSE

FLEX-DIRECTION: COLUMN

FLEX-DIRECTION: COLUMN REVERSE

FLEX - WRAP

Utilizar o flex-wrap: restringe os itens do FLEX a não quebrarem o container e se moverem para baixo. Na primeira sessão falando sobre o display: flex damos o exemplo que os itens após passar do tamanho do container, os ignora e continua sendo adicionado um na frente do outro. Aqui ocorre algo diferente.

EXEMPLO:

FLEX-WRAP: WRAP

textoTeste
textoTeste
textoTeste
textoTeste
textoTeste

FLEX-WRAP: WRAP-REVERSE

textoTeste
textoTeste
textoTeste
textoTeste
textoTeste

Flex - flow

Atalho para unir as duas opções acima em uma mesma linha.

EXEMPLO:

FLEX - FLOW: ROW NOWRAP (diminui o tamanho do container para mostrar que o há o estouro da delimitação)


FLEX - FLOW: ROW WRAP (o mesmo container acima, porém utilizando o flex-flow: now wrap)


É ainda possível utilizar flex-flow: column wrap e flex-flow: column nowrap
