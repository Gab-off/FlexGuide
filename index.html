Flex - Direction

Utilizar o flex-direction: define a direção com que os elementos irão seguir.

EXEMPLO:

FLEX-DIRECTION: ROW (modo padrão quando se dá um display: flex)

FLEX-DIRECTION: ROW-REVERSE

FLEX-DIRECTION: COLUMN

FLEX-DIRECTION: COLUMN REVERSE

FLEX - WRAP

Utilizar o flex-wrap: restringe os itens do FLEX a não quebrarem o container e se moverem para baixo. Na primeira sessão falando sobre o display: flex damos o exemplo que os itens após passar do tamanho do container, os ignora e continua sendo adicionado um na frente do outro. Aqui ocorre algo diferente.
